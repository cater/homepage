### 手动部署

* **安装** [node.js](https://nodejs.org/zh-cn/) **环境**

  > node >= 18.19.1  
  > npm > 10.2.4
  
* 然后以 **管理员权限** 运行 `cmd` 终端，并 `cd` 到 项目根目录
* 在 `终端` 中输入：

```bash
# 安装 pnpm
npm install -g pnpm

# 安装依赖
pnpm install

# 预览
pnpm dev

# 构建
pnpm build
```

> 构建完成后，也可使用 `Vercel` 等托管平台一键导入并自动部署
